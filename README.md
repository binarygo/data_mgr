# data_mgr

* MNIST Dataset (mnist_data_mgr.py)
- Hardcoded data downlink.
- Can be used directly.

* Anime Face Dataset (anime_face_data_mgr.py)
- Need to download data from
  http://www.nurs.or.jp/~nagadomi/animeface-character-dataset/data/animeface-character-dataset.zip
- Extract all files in .. and rename the top level directory to "anime_face".
